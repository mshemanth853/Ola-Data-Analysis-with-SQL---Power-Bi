#  OLA Rides Data Analysis Dashboard  

##  Project Overview  
Developed an interactive **Power BI dashboard** by analyzing and visualizing **OLA ride data**.  
The dataset was cleaned, structured, and processed using **SQL**, followed by creating **five insightful and interactive Power BI pages** to analyze booking trends, cancellations, revenue patterns, and customer behavior.

##  Key Insights  
- Total of **1 lakh rides** analyzed.  
- **62%** of bookings were successful, while **38%** were cancelled or incomplete.  
- **Cash** and **UPI** were the most preferred payment methods.  
- Majority of cancellations were due to **driver unavailability** or **customer plan changes**.  
- Ratings for both customers and drivers averaged around **4.0 / 5**, indicating strong service quality.  
- Consistent weekday activity pattern observed in bookings.
